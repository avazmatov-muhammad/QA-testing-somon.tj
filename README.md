# 🧪 QA Testing — Somon.tj

---

## 📌 О проекте

Данный проект представляет собой пример ручного тестирования (Manual Testing) веб-сайта Somon.tj.

Цель проекта — продемонстрировать базовые навыки QA:
- анализ функциональности
- тест-дизайн (Test Design)
- написание тестовой документации
- поиск и оформление багов

---

## 🎯 Цель тестирования (Goal)

- Проверка основного функционала сайта
- Проверка пользовательского интерфейса (UI)
- Проверка граничных значений и некорректных данных (BVA, Negative Testing)
- Оценка базовой производительности (Performance)

---

## 🌐 Объект тестирования (System Under Test)

- Website: https://somon.tj

---

## 🧪 Область тестирования (Scope)

В рамках проекта были протестированы следующие модули:

- Поиск объявлений (Search)
- Фильтрация (Filters)
- Просмотр объявления (Ad Page)
- Пользовательский интерфейс (UI)
- Адаптивность (Mobile view)

---

## 🧠 Используемые техники (Test Design)

- Equivalence Partitioning
- Boundary Value Analysis (BVA)
- Negative Testing
- Exploratory Testing

---

## 🛠 Используемые инструменты (Tools)

- Google Chrome
- Chrome DevTools
- GitHub
